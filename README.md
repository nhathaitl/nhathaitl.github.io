## 💻 Local Development

If you want to run this project locally to preview changes or write draft posts, follow these steps:

### Prerequisites
Make sure you have Ruby and Bundler installed on your system.

### Steps
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/nhathaitl/nhathaitl.github.io.git
    cd nhathaitl.github.io
    ```
2.  **Install dependencies:**
    ```bash
    bundle install
    ```
3.  **Run the local Jekyll server:**
    ```bash
    bundle exec jekyll serve
    ```
4.  Open your browser and navigate to `http://127.0.0.1:4000`.

## ✍️ Writing New Posts

All blog posts are written in Markdown and live in the `_posts/` directory.

File names must follow the format: `YYYY-MM-DD-title.md` (e.g., `2026-06-20-automotive-ethernet-basics.md`).

Every post should start with front matter:
```yaml
---
title: Post Title
date: YYYY-MM-DD HH:MM:SS +/-TTTT
categories: [Category, Subcategory]
tags: [tag1, tag2]
---
```
